AUTO SORT: Sort tabs automatically based on certain criteria

TAB SUSPENDER: Suspend inactive tabs after X minutes

WAIT TO LOAD: Don't load newly opened tabs until selected

OPEN MULTIPLE LINKS: Open multiple links in one click

TAB RELOADER: Reloads a given tab every X minutes

The ALL WINDOWS switch in main menu dictates if the buttons underneath it affect only your current window or all windows.

The WAIT TO LOAD switch in OPEN MULTIPLE LINKS overrides the main WAIT TO LOAD setting and ignores included/excluded domains.

The SORT & SUSPEND TABS buttons use the settings from AUTO SORT and TAB SUSPENDER respectively.

With tools that let you include / exclude, you need to provide domains. If you provide links, they will be converted to domains once you apply.
Example domain: “google.com”

Include: apply only to the sites you provide
Exclude: apply to all sites but the ones you provide

Whenever an APPLY button is present, you must click on it for the new settings to take effect. Once applied, invalid settings will be reverted to what they were prior. If present and necessary, links will be converted and duplicates will be removed.

Number input fields where you need to enter the number of minutes can be decimals and can also start with 0 (ex. 0.15).

When you mute a tab with this extension, you won't be able to right click unmute it from its menu. You will have to unmute it using this extension. This appears to be some sort of a bug (or a safety feature) in the Chrome browser.
